# Shopping Website

This is a modern Shopping Website built with [Next.js](https://nextjs.org), [Prisma](https://www.prisma.io/), and [TypeScript](https://www.typescriptlang.org/).

## Features

- **Modern UI/UX**: Built with a focus on aesthetics and user experience.
- **Product Management**: Browse and view product details.
- **Cart System**: Add products to cart and manage orders.
- **Database Integrated**: Uses Prisma ORM for database interactions.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- **Framework**: [Next.js 14+](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Database**: [PostgreSQL](https://www.postgresql.org/) (via Prisma)
- **Styling**: Tailwind CSS (assumed based on standard Next.js setup) / CSS
- **ORM**: [Prisma](https://www.prisma.io/)
