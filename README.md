## üåü About The Project

**Elegant Fashion** is a cutting-edge shopping website designed to provide a seamless and visually stunning e-commerce experience. Built with performance and scalability in mind, it leverages the power of Next.js 16 and React 19 to deliver lightning-fast page loads and smooth interactions.

### Key Highlights:
*   **Modern Design**: Aesthetic UI crafted with Tailwind CSS v4 and Framer Motion for fluid animations.
*   **Robust Backend**: Powered by Prisma ORM for type-safe database interactions with PostgreSQL.
*   **Developer Experience**: strict TypeScript integration ensuring code reliability and maintainability.

## üöÄ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or yarn
*   PostgreSQL database

### Installation

1.  **Clone the repo**
    ```sh
    git clone https://github.com/HisyamAlammar/Shopping-Website.git
    cd Shopping-Website
    ```

2.  **Install NPM packages**
    ```sh
    npm install
    ```

3.  **Set up Environment Variables**
    Create a `.env` file in the root directory and add your database connection string and other secrets.
    ```env
    DATABASE_URL="postgresql://username:password@localhost:5432/mydb?schema=public"
    ```

4.  **Run Database Migrations**
    ```sh
    npx prisma migrate dev
    ```

5.  **Run the Development Server**
    ```sh
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
